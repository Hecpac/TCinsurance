# Meta Ads — TC Insurance Campaign Framework

Structure for running Meta (Facebook/Instagram) paid campaigns for TC Insurance.

## Structure

```
meta-ads/
  campaign-structure/   CBO per product, single ad set start, new ads flow, ABO testing
  creative-testing/     Monthly concepts, asset volume, concept gaps, spend allocation
  creative-formats/     Static (70%), yapper videos, partnership ads, motion graphics
  audience-strategy/    Broad targeting, exclusions, creative-as-targeting, reach goals
  health-metrics/       CPMr, net new reach %, creative win rate, cost per 1k reached
  measurement/          Blended revenue, third-party attribution, ROAS, contribution margin
  signals-to-watch/     CPMr climbing, reach declining, ROAS+CPMr warning, frequency cap
```

## Quick Reference

| Metric | Healthy Range |
|--------|--------------|
| Creative win rate | 10-20% |
| Net new reach | > 70% |
| Frequency (7d) | < 2.5 |
| Static ad ratio | ~70% of output |
| Monthly concepts | 8-10 |
| Monthly assets | 40-50 |
