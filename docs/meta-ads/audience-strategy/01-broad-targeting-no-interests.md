# Broad Targeting (No Interests)

Do NOT add interest targeting. Let the creative do the targeting.

## Why Broad

- Meta's algorithm is better at finding buyers than manual interest targeting
- Interest audiences are noisy and overlap heavily
- Broad gives Meta maximum room to optimize
- Lower CPMs due to larger auction pool

## TC Insurance Targeting Setup

| Parameter | Setting |
|-----------|---------|
| Location | Dallas-Fort Worth metro (25-mile radius) |
| Age | 25-65 |
| Gender | All |
| Language | Spanish + English |
| Interests | NONE |
| Behaviors | NONE |
| Lookalikes | NONE (unless testing in ABO) |

## The Only Exception

- Retargeting campaigns (warm audiences) use custom audiences
- But even retargeting should be broad within the custom audience
