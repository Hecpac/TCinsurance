# Static Ads (70% of Output)

Static image ads are the foundation. They're fast to produce, easy to test, and perform consistently.

## Why 70%

- Lowest production cost per asset
- Fastest to iterate (new concept in 30 minutes)
- Work across all placements (feed, stories, reels, right column)
- Easy to A/B test copy and visual independently

## TC Insurance Static Ad Types

| Type | Example | Best For |
|------|---------|----------|
| Comparison table | Bronze vs. Silver vs. Gold | Logical buyers |
| Price callout | "Desde $0/mes con subsidio" | Price-sensitive |
| Testimonial card | Client quote + photo | Social proof |
| Deadline banner | "Enrollment ends Jan 15" | Urgency |
| Checklist | "5 things to know before choosing" | Education |
| Before/after | "Without agent" vs. "With TC Insurance" | Value prop |

## Design Rules

- Dark background (#111318) with blue accents (#A4C9FE) — match website
- Zero border-radius (match brand)
- Clear CTA text in Spanish
- Tatiana's photo when possible (builds trust)
- Large, readable text (people scroll fast)
