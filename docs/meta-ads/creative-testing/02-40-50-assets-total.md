# 40-50 Assets Total per Month

Each concept produces multiple assets (sizes, variations, copy tweaks).

## Math

- 8-10 concepts x 4-5 variations each = 40-50 assets
- Variations include: different sizes (1:1, 4:5, 9:16), copy changes, CTA changes

## Asset Checklist per Concept

- [ ] 1:1 square (feed)
- [ ] 4:5 vertical (feed, preferred)
- [ ] 9:16 stories/reels
- [ ] 2-3 copy variations (short, medium, long)
- [ ] 2 CTA variations (e.g., "Agenda tu consulta" vs. "Llama ahora")

## Production Notes

- Batch production: create all variations of one concept before moving to the next
- Use Canva templates for static ads (maintain TC Insurance brand: dark bg, blue accents)
- Keep source files organized by concept name and date
