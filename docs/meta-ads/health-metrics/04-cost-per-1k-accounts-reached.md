# Cost per 1,000 Accounts Reached

How much it costs to reach 1,000 unique people (not impressions).

## Formula

```
Cost per 1k Reached = (Total Spend / Unique Reach) x 1,000
```

## Why This Matters

- More meaningful than CPM (which counts repeat impressions)
- Tells you the actual cost of expanding your audience
- Rising cost per 1k reached = audience saturation

## Benchmarks (Insurance, DFW Market)

| Range | Verdict |
|-------|---------|
| $3-$8 | Excellent efficiency |
| $8-$15 | Normal for insurance vertical |
| $15-$25 | Getting expensive, check creative fatigue |
| $25+ | Saturated, need creative refresh or audience expansion |

## How to Track

1. Ads Manager > Customize Columns
2. Add: Amount Spent, Reach
3. Calculate: (Spend / Reach) x 1,000
4. Track weekly, compare trends
