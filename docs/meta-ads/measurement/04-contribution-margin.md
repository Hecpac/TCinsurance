# Contribution Margin (Actual Profit)

What's left after subtracting ad spend and cost of servicing from revenue.

## Formula

```
Contribution Margin = Revenue - Ad Spend - Cost of Service
```

## TC Insurance Calculation

| Item | Value |
|------|-------|
| Avg first-year commission per policy | $300 |
| Avg renewal commission (years 2-5) | $150/year |
| Lifetime value per client (5 years) | $900 |
| Cost to acquire (target CPA) | $80 |
| Cost to service (time, tools) | $20 |
| Contribution margin per client | $800 |
| Contribution margin % | 89% |

## Why This Matters

- A $80 CPA looks expensive until you realize LTV is $900
- Contribution margin tells you if scaling is profitable
- Track monthly to ensure ad costs aren't growing faster than revenue

## Break-Even Analysis

```
Break-even CPA = First-year commission = $300
Target CPA = 25-30% of first-year commission = $75-$90
Stretch CPA = Considering LTV = up to $150
```

## Rules

- Never optimize for CPA alone — optimize for contribution margin
- A higher CPA with better-quality leads can be more profitable
- Track policy activation rate (lead → actual policy) separately
