# Single Ad Set per Campaign (Start Here)

Start every new campaign with ONE ad set. Only add more when you have a reason.

## Why Single Ad Set

- Consolidates learning data into one audience pool
- Exits learning phase faster (50 events/week threshold)
- Avoids audience overlap and self-competition
- Simplifies budget allocation

## When to Add a Second Ad Set

- You have a distinctly different audience hypothesis to test
- The first ad set has been stable for 2+ weeks
- You have enough budget to support 50 events/week per ad set
