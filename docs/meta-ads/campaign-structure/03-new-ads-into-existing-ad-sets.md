# New Ads into Existing Ad Sets

Never create a new ad set just to test a new creative. Add new ads to existing ad sets.

## Why

- Preserves accumulated learning data
- Lets Meta compare new vs. existing creatives fairly
- Avoids resetting the learning phase
- Keeps audience targeting consistent

## Process

1. Create new ad creative (image/video + copy)
2. Add it to the existing ad set alongside current winners
3. Let Meta allocate spend naturally (don't force equal distribution)
4. Wait 3-5 days before evaluating
5. Kill losers (CTR < 1%, CPL > 2x target) after sufficient impressions (1,000+)

## Frequency

- Add 2-3 new ads per ad set every 1-2 weeks
- Never have more than 6 active ads per ad set
- Pause underperformers before adding new ones
