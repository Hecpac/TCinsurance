# Separate ABO for Testing Only

Ad Set Budget Optimization (ABO) campaigns are reserved exclusively for creative testing.

## When to Use ABO

- Testing a new creative concept before putting it into a CBO campaign
- Comparing two radically different approaches (e.g., Spanish-only vs. bilingual)
- A/B testing landing pages
- Testing a new audience hypothesis in isolation

## Setup

- Budget: 10-15% of total ad spend
- Duration: 3-7 days per test
- Equal budget per ad set (forced equal distribution)
- 1-2 ads per ad set maximum
- Clear hypothesis documented before launch

## Graduation Criteria

- CTR > 1.5%
- CPL within target range
- Consistent performance over 3+ days
- Move winning creative to main CBO campaign
